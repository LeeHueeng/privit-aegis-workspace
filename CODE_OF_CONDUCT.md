# Code of Conduct

Privit Aegis is a security project, so the bar for respectful collaboration is
high. Be direct, evidence-based, and kind.

## Expected Behavior

- Treat contributors and reporters with respect.
- Assume security findings may be sensitive.
- Discuss impact, evidence, and scope without personal attacks.
- Help newcomers understand safe testing boundaries.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language
- Publishing credentials, private data, or exploit details outside approved
  channels
- Encouraging destructive testing against unauthorized targets
- Deliberately bypassing project safety controls

## Reporting

For security-sensitive issues, follow [`SECURITY.md`](./SECURITY.md). For
community issues, open a private maintainer discussion or contact the repository
owner through GitHub.
