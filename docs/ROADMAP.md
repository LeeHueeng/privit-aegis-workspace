# Roadmap

This roadmap is intentionally public-facing. It helps visitors understand what
is already useful, what is planned, and where contributions can make the project
more valuable.

## Now

- Local web console for Aegis workflows
- Passive site discovery and auth-surface inventory
- Localized HTML reports in Korean, English, Japanese, and Chinese
- Passive penetration/security testing report with pass criteria
- AI provider settings for Codex, Gemini, Claude, local AI, and direct API use
- GitHub Actions gate with AIGate, hardening checks, and public CLI install

## Next

- More examples of safe findings and fixed findings
- Exportable report bundle for review meetings
- GitHub Pages screenshots generated from current report samples
- Additional i18n review pass for Japanese and Chinese wording
- Safer opt-in active checks gated by explicit scope flags

## Good First Contributions

- Improve wording in localized Markdown files
- Add passive-only detection documentation
- Add small fixtures for report localization
- Improve GitHub Pages examples and diagrams
- Add tests for docs links and command references

## Non-Goals

- No destructive scanning by default
- No credential stuffing, brute force, or live payment testing
- No testing of third-party systems without written authorization
